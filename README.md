# GPT-like Transformer

Это готовая реализация GPT-подобной языковой модели (трансформер), написанная с нуля на PyTorch.

## Возможности
- Архитектура: Decoder-only Transformer
- Позиционные эмбеддинги: Rotary Positional Embeddings (RoPE)
- Активация: SwiGLU
- Attention: Flash Attention v2 (PyTorch 2.0+)
- Токенизатор: BPE с нуля
- Оптимизатор: 8‑битный AdamW (bitsandbytes)
- Функции генерации: top‑p, top‑k, температура, repetition penalty

## Состав репозитория
- `config.py` – настройки модели
- `model.py` – архитектура трансформера
- `tokenizer.py` – BPE‑токенизатор
- `train.py` – обучение модели
- `generate.py` – генерация текста
